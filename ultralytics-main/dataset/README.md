## 拍照要求

### 每种物品的拍摄方案（每件物品至少拍 50 张）

| 角度 | 张数 | 说明 |
|------|------|------|
| 正面 0° | 10张 | 物品正对相机 |
| 左侧 30° | 10张 | 物品旋转30° |
| 右侧 30° | 10张 | 物品旋转-30° |
| 俯视 45° | 10张 | 相机略高 |
| 随机角度 | 10张 | 任意姿态（模拟比赛实际摆放） |

### 变化条件（每件物品至少覆盖）

| 条件 | 张数 | 说明 |
|------|------|------|
| 纯色背景 | ~20张 | 白/黑/蓝/红/绿 各若干 |
| 混合背景 | ~15张 | 条纹/格子/花色桌布 |
| 自然光 | ~15张 | 室内灯光，无特殊光源 |
| 黄光（7W台灯） | ~10张 | 距离30-50cm照射 |
| 白光（7W台灯） | ~10张 | 距离30-50cm照射 |

### 距离变化
- 0.7m（近距）~10张
- 1.2m（中距）~10张
- 1.8m（远距）~10张
- 其他距离 ~20张

## 目录结构

```
dataset/
├── images/
│   ├── train/          ← 训练集图片（~80%）
│   │   ├── CA001_01_0deg_white_natural_1.0m.jpg
│   │   ├── CA001_01_30deg_blue_white_1.5m.jpg
│   │   └── ...
│   └── val/            ← 验证集图片（~20%）
│       ├── CA001_03_45deg_red_natural_0.8m.jpg
│       └── ...
├── labels/
│   ├── train/          ← 训练集标签（YOLO格式.txt）
│   │   ├── CA001_01_0deg_white_natural_1.0m.txt
│   │   └── ...
│   └── val/            ← 验证集标签
│       └── ...
└── README.md           ← 本文件
```

### 文件命名规范

```
{类别编号}_{物品序号}_{角度}_{背景}_{光照}_{距离}.jpg

示例: CA001_03_30deg_white_yellow_1.2m.jpg
      CB003_01_0deg_mixed_natural_0.7m.jpg
```

### YOLO标签格式（每张图片对应一个同名.txt）

```
{class_id} {x_center} {y_center} {width} {height}

class_id:  0=CA001, 1=CA002, ..., 15=CD004, 16=W001, 17=W002
x_center:  标注框中心x / 图片宽度  （0~1之间的浮点数）
y_center:  标注框中心y / 图片高度  （0~1之间的浮点数）
width:     标注框宽度 / 图片宽度
height:    标注框高度 / 图片高度
```

### 标注工具推荐

- **LabelImg** (免费) — https://github.com/HumanSignal/labelImg
- **Roboflow** (在线) — https://roboflow.com （免费额度足够）
- **CVAT** (在线) — https://cvat.ai
